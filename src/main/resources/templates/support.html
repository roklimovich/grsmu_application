GRSMU File Server

Ответы на часто задаваемые вопросы

❓- Зачем нужен сайт?

✅ - Сайт создается для сохранения и постоянного доступа к материалам с разных курсов, которые могут помочь студентам для подготовки к занятиям, экзаменам и прочему. Вместо серфинга и поиска необходимой информации в гугле все возможно будет получить на одном сайте.

❓- Почему нельзя создать Google-Диск для этого?

✅ - Теоретически это возможно и будет хорошим решением. Однако, Гугл-Диск не дает возможности в написании нового функционала и подключения сторонних сервисов по типу ChatGPT и прочих. Сайт дает возможность для масштабирования базового функционала. На начальном этапе - это обычный сайт для того, чтобы делиться материалом. В дальнейшем функционал будет расширяться и следующим пунктом после запуска приложения в продакшн - будет создание тестов, благодаря которым можно будет подготовиться к экзамену, либо же закрепить изученный материал. Если вы хотите поделиться материалами или предложить правки, напишите в Telegram: @grsmu_web_server.

❓- Прозрачный ли сайт?

✅ - Да, код сайта, как и весь материал будет находиться в открытом GitHub репозитории.

❓- Откуда берется материал на сайте?

✅ - Сайт будет пополняться материалами студентов, которые будут прилетать в сообщения данной группы. Идея такова:

Cтудент присылает материал со своего курса по определенному предмету/курсу в личные сообщения этого канала -> файлы автоматически проходят модерацию -> загружаются на хостинг -> через несколько минут приложение будет видеть новые файлы и отображать на сайте.
